Smart WebView Pro

What is Smart WebView?

Smart WebView is hybrid project designed to develop advanced android applications using basic web technologies. It works as a pipeline between webpages and android native features, creating and fetching requests on behalf of both technologies.

Note: SWV 2.0 is currently under tests and will be released by November, 2020 end. You can pre-order below and help speed up the production.

Pro Features

Core
  • Live GPS Location Tracking
  • File Upload
  • Camera Photo Processing
  • Firebase Push Notifications
  • Google AdMob
  • Navigation Drawer
  • Search Bar
  • Action Menu
  • Chrome Tab
  • Handling Data Sharing
  • Different Layouts Selection
Included Plugins
  • Google Login
  • PaperlessQL Database
  • Send to Server (S2S)
  • QReader for Barcodes and QR
  • Background Player for music
  • Advanced Firebase Messaging + Script
  • Google Vision API (preview)
